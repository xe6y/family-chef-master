# 家庭厨师微信小程序

## 项目简介
一个专为家庭设计的智能点餐和菜品管理小程序

## 技术架构
- **后端**: Go + Gin + GORM + MySQL
- **前端**: uni-app + Vue3 + TypeScript
- **数据库**: MySQL 8.0
- **缓存**: Redis
- **文件存储**: 腾讯云COS
- **消息推送**: 微信小程序订阅消息

## 开发环境要求
- Go 1.21+
- Node.js 18+
- MySQL 8.0+
- Redis 6.0+
- 微信开发者工具

## 快速开始
1. 克隆项目
2. 配置数据库和Redis
3. 启动后端服务
4. 启动前端开发服务器
5. 在微信开发者工具中打开前端项目
